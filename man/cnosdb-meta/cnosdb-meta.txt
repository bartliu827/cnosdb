cnosdb-meta(1)
===============

NAME 
----
cnosdb-meta - Store database metadata

SYNOPSIS
--------

[verse]
cnosdb-meta [command] [flags]
cnosdb-meta [command]

DESCRIPTION
-----------
'cnosdb-meta' stores database metadata.

COMMANDS
--------
The 'cnosdb-meta' command starts the CnosDB meta node.

config::
  display the default configuration.

help::
  Help about any command.

run::
  Runs the CnosDB Meta Server.

version::
  Displays the CnosDB meta version.
